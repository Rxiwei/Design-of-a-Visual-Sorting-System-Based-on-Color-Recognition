# Design-of-a-Visual-Sorting-System-Based-on-Color-Recognition
Color-Based Visual Sorting System using K210 + STM32 for automatic material recognition and pneumatic sorting.

一： Project Overview
This is a course design project that implements real-time visual recognition of red, green, and blue objects based on the K210 development board. The recognition results are visually marked on an LCD screen, and the recognition status is transmitted to external devices (such as STM32F103C8T6) via the UART serial port.

二：Experimental Equipment
K210 Development Board
STM32F103C8T6
LCD Display
Camera Module

三： Experimental Environment
MaixPy IDE
Keil uVision5 (Keil5)

四： Visual Recognition Algorithm
Blob Detection

一：项目概述
